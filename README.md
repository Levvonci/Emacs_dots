# Emacs_dots
Recentemente ho iniziato ad interessarmi al linguaggio lisp proprio tramite emacs.
Utilizzeró questo repo per mantenere aggiornati i miei dots

Il file da tenere in considerazione é **dotemacs.org** dove risiede effettivamente la configurazione.
L'init é il **.el** symlinkato ad un file nella mia `/home` 

La configurazione per adesso é abbastanza minimale e standard, probabilmente con il tempo si evolverá in qualcosa di piú strutturato e sicuramente piú pulito.
